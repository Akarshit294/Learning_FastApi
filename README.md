# Learning FastAPI

Personal repo for learning FastAPI by following the [official docs](https://fastapi.tiangolo.com/learn/).

## Contents
- `notes.md` — running notes on concepts covered
- Topic folders (`01_basics/`, etc.) added as learning progresses

## Setup
```bash
uv init
uv add "fastapi[standard]"
uv run uvicorn main:app --reload
```
